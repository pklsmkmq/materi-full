## 1. Aktivasi Authentication di Firebase Console

Sebelum menulis kode, kita harus memberi tahu Firebase metode login apa saja yang kita izinkan di aplikasi kita.

**Langkah-langkah:**

1. Buka kembali dashboard project Firebase kalian di browser.

2. Di menu sebelah kiri, cari bagian **Build**, lalu klik **Authentication**.

3. Klik tombol **Get Started** (Mulai).

4. Pilih tab **Sign-in method** (Metode login).

5. Klik **Email/Password**, aktifkan *toggle* "Enable" (yang atas saja), lalu klik **Save**.

6. Klik tombol **Add new provider**, pilih **Google**, aktifkan "Enable", masukkan email *support* kalian, lalu klik **Save**.

## 3. Logika Autentikasi di `AuthProvider`

Dalam Flutter, sangat disarankan untuk memisahkan antara *tampilan* (UI) dan *logika* (Proses). Kita akan menaruh semua logika Firebase Auth di dalam file Provider.

Buka file `lib/providers/auth_provider.dart` dan perhatikan logika dasar berikut:

```dart
import 'package:firebase_auth/firebase_auth.dart';
import 'package:flutter/material.dart';

class AuthProvider with ChangeNotifier {
  final FirebaseAuth _auth = FirebaseAuth.instance;
  
  // Variabel untuk menyimpan status user saat ini
  User? _user;
  User? get user => _user;

  AuthProvider() {
    // Memantau perubahan status login secara realtime (Stream)
    _auth.authStateChanges().listen((User? currentUser) {
      _user = currentUser;
      notifyListeners(); // Beri tahu UI kalau status berubah
    });
  }

  // Fungsi Login dengan Email
  Future<void> loginWithEmail(String email, String password) async {
    try {
      await _auth.signInWithEmailAndPassword(email: email, password: password);
    } catch (e) {
      throw Exception('Gagal login: ${e.toString()}');
    }
  }

  // Fungsi Logout
  Future<void> logout() async {
    await _auth.signOut();
  }
  
  // (Fungsi Register dan Login Google ditambahkan di sini)
}
```

**Penjelasan:** Fungsi `authStateChanges().listen(...)` adalah bagian terpenting. Fungsi ini akan terus memantau: *"Apakah ada user yang sedang login saat ini?"*

## 4. Proteksi Halaman (Route Guard) dengan GoRouter

Apa gunanya sistem login jika siswa yang belum login tetap bisa membuka halaman Dashboard secara diam-diam? 

Kita harus "mengunci" halaman tersebut. Buka file `lib/core/routes/app_router.dart`. Di sinilah `GoRouter` bekerja sama dengan Firebase Auth.

```dart
import 'package:go_router/go_router.dart';
import 'package:firebase_auth/firebase_auth.dart';
// import screen kalian di sini...

final GoRouter appRouter = GoRouter(
  initialLocation: '/login', // Halaman pertama kali dibuka
  
  // FUNGSI PENGECEKAN (REDIRECT)
  redirect: (context, state) {
    // Cek apakah ada user yang sedang login di Firebase
    final bool isLoggedIn = FirebaseAuth.instance.currentUser != null;
    
    // Cek apakah user sedang berada di halaman login
    final bool isGoingToLogin = state.uri.toString() == '/login';

    // Jika belum login, dan mencoba masuk ke halaman selain login, lempar kembali ke '/login'
    if (!isLoggedIn && !isGoingToLogin) {
      return '/login';
    }

    // Jika sudah login, tapi mencoba buka halaman login, lempar ke '/dashboard'
    if (isLoggedIn && isGoingToLogin) {
      return '/dashboard';
    }

    // Biarkan jalan normal
    return null;
  },

  routes: [
    GoRoute(
      path: '/login',
      builder: (context, state) => const LoginScreen(),
    ),
    GoRoute(
      path: '/dashboard',
      builder: (context, state) => const DashboardScreen(),
    ),
  ],
);
```

Dengan kode di atas, jika `FirebaseAuth.instance.currentUser` kosong (null), aplikasi tidak akan pernah mengizinkan user masuk ke `/dashboard`.

## 5. Menerapkan di UI (Tampilan Login)

Sekarang, di file `login_screen.dart`, kalian tinggal memanggil fungsi dari provider saat tombol diklik.

Contoh pemanggilan fungsi pada tombol Login:

```dart
ElevatedButton(
  onPressed: () async {
    try {
      // Memanggil fungsi login dari AuthProvider
      await context.read<AuthProvider>().loginWithEmail(
        emailController.text, 
        passwordController.text
      );
      // Jika berhasil, GoRouter akan otomatis melempar ke Dashboard berkat fungsi redirect di atas!
    } catch (e) {
      // Tampilkan error menggunakan SnackBar
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(content: Text(e.toString())),
      );
    }
  },
  child: const Text('Login'),
)
```
