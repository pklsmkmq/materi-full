# Mobile (Flutter)

Materi membangun aplikasi mobile dengan Flutter.

## Daftar Materi
- Pengenalan Dart
- Widget dasar
- Layout (Row/Column/Stack)
- State management (konsep)
- Konsumsi API
- Build APK / release

!!! tip "Target"
    Setelah bab ini, kamu bisa membuat aplikasi mobile sederhana dan mengambil data dari API.
