# Build APK & Mengganti Icon Aplikasi Flutter

## Tujuan Pembelajaran
Siswa mampu:  
1. Melakukan build APK Flutter (debug & release).  
2. Memahami perbedaan APK debug dan release.  
3. Mengganti icon aplikasi menggunakan gambar sendiri.  
4. Menggunakan website / package untuk generate icon.  
5. Menyiapkan aplikasi agar siap diinstal di HP Android.  

---

## A. Build APK Flutter

### Build APK Debug
```bash
flutter build apk --debug
```

File hasil:
```
build/app/outputs/flutter-apk/app-debug.apk
```

### Build APK Release
```bash
flutter build apk --release
```

File hasil:
```
build/app/outputs/flutter-apk/app-release.apk
```

---

## B. Mengganti Icon Aplikasi

### Opsi Website
1. Siapkan icon PNG 512x512  
2. Upload ke:  
   - https://romannurik.github.io/AndroidAssetStudio/icons-launcher.html  
   - https://www.appicon.co  
   - https://icon.kitchen  
3. Download dan extract  
4. Ganti file di:  
```
android/app/src/main/res/mipmap-*
```

Nama file:
```
ic_launcher.png
```

---

### Opsi Package flutter_launcher_icons

`pubspec.yaml`:
```yaml
dev_dependencies:
  flutter_launcher_icons: ^0.13.1

flutter_icons:
  android: true
  image_path: assets/icon/app_icon.png
```

Generate:
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

---

## Tugas  
1. Build APK release  
2. Ganti icon aplikasi  
3. Install di HP/Emulator
