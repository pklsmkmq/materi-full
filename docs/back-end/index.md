# Modul Back End (NestJS)

## Gambaran Umum

Back End adalah bagian aplikasi yang **mengelola logika, data, dan komunikasi** antara Front End dan database.
Pengguna tidak melihat Back End secara langsung, tetapi seluruh fitur aplikasi bergantung padanya.

Contoh tugas Back End:  
- Mengelola data pengguna  
- Menyimpan & mengambil data dari database  
- Membuat API  
- Mengatur autentikasi & otorisasi  
- Menjaga keamanan data

Pada modul ini, framework Back End yang digunakan adalah **NestJS**.

---

## Kenapa Menggunakan NestJS?

NestJS adalah framework Back End modern berbasis **Node.js** yang:  
- Ditulis menggunakan **TypeScript**  
- Menggunakan konsep **OOP (Object Oriented Programming)**  
- Struktur kode rapi dan terorganisir  
- Banyak digunakan di industri

NestJS cocok untuk:  
- Aplikasi skala kecil hingga besar  
- Tim developer  
- Project yang membutuhkan struktur jelas

---

## Kenapa Harus Belajar TypeScript Dulu?

NestJS **100% berbasis TypeScript**.  
Tanpa memahami TypeScript, belajar NestJS akan terasa sulit dan membingungkan.

TypeScript diperlukan karena:  
- NestJS menggunakan **class**  
- NestJS menggunakan **interface**  
- NestJS menggunakan **decorator**  
- Data harus memiliki **struktur yang jelas**

Dengan menguasai TypeScript terlebih dahulu, kamu akan:  
- Lebih mudah memahami Controller, Service, dan Module  
- Tidak bingung dengan error tipe data  
- Lebih cepat memahami alur kerja NestJS  

> **Kesimpulan:**  
> TypeScript adalah fondasi utama sebelum masuk ke NestJS.
